# Core Club ⚡️

**Build better days** — تطبيق ويب لتتبّع أنشطتك، سعراتك، ماءك وقياساتك في مكان واحد.

## المميزات

- تسجيل دخول محلي بأكثر من حساب (اسم مستخدم + رمز سري) — البيانات تنحفظ على جهاز المستخدم فقط
- جدول أنشطة ذكي: النشاط الفائت ينتقل لليوم اللي بعده تلقائيًا، مع إمكانية تحديد نشاط اليوم يدويًا
- مقياس انضباط لآخر 4 أسابيع
- تتبّع سعرات بألوان (🟢 ضمن الهدف / 🔴 تجاوز / 🟡 أقل) وحساب تلقائي بمعادلة Mifflin-St Jeor
- عدّاد أكواب ماء حسب الجنس
- وزن وقياسات مع فرق آخر قياس (🔴 زاد / 🟢 نزل / 🟡 ثابت)
- ملخص أسبوعي وشهري وسنوي

## طريقة الرفع على GitHub Pages

1. أنشئ مستودع (Repository) جديد في GitHub — سمّه مثلًا `core-club` وخله **Public**
2. ارفع ملف `index.html` (زر **Add file → Upload files**)
3. من **Settings → Pages**: تحت Branch اختر `main` و `/ (root)` ثم **Save**
4. انتظر دقيقة، وموقعك بيكون على:
   `https://اسم-حسابك.github.io/core-club/`

## ملاحظات تقنية

- ملف واحد بدون أي اعتماديات build — HTML/CSS/JS خام
- البيانات تُحفظ في `localStorage` بمتصفح المستخدم (لا يوجد سيرفر، ولا تنتقل البيانات بين الأجهزة)
- يدعم RTL بالكامل وخط IBM Plex Sans Arabic
